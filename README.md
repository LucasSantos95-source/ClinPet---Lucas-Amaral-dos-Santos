# Sistema de Clínica Veterinária (Clínica Pet)

## Status do Projeto
Em desenvolvimento

## Tecnologias Utilizadas
- Java
- Java Swing
- JDBC
- MySQL
- Apache NetBeans

## Desenvolvedor
- Lucas Amaral dos Santos

## Objetivo do Software
O sistema tem como objetivo gerenciar atendimentos de uma clínica veterinária, permitindo o controle de consultas de forma simples e organizada.

## Funcionalidades do Sistema
- Agendar consultas
- Listar consultas
- Cadastrar pets
- Cadastrar tutores
- Registrar veterinário responsável
- Encerrar o sistema
